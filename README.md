# DigitalClapperboard
This Repo contains a renderer for MAX72xx led matrices for use with an arduino style microcontroller. The code contains also the specifics for use with a switch to alter the display's Text. 

A list of all materials used is in the code comments.

Additionally you'll need a 9V power supply. I just used a battery. 

I hand soldered the connector board. But you can also just use a bread board or make a PCB.

The schematics are in the src folder.
